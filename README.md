
# Research Site Template (Clean-Room)

This is an **original, clean-room** static website template for a research & analytics company.
It does **not** copy any proprietary code, text, or assets from any third-party website. You can
customize branding, content, and colors to match your company.

## Quick start

1. Open `index.html` in your browser.
2. Edit text and links in the HTML files.
3. Update colors in `assets/css/styles.css`.
4. Replace placeholder images in `assets/img/` with your own.

## Deploy

This is a purely static site and can be hosted on any static host (Azure Static Web Apps, Netlify,
GitHub Pages, Vercel, S3+CloudFront, etc.). Upload the folder as-is.

## GA4 (optional)

In `assets/js/main.js`, uncomment the GA4 section and set your Measurement ID.

## Accessibility & SEO

- Semantic HTML5 structure and landmarks.
- Mobile-friendly responsive layout.
- Open Graph tags + JSON-LD Organization schema (edit with your details).
